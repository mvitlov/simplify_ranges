# simplify_ranges

> Merge and normalize adjacent or overlapping integer ranges

## Usage

```dart
import 'package:simplify_ranges/simplify_ranges.dart';

const ranges = [[1, 2], [3, 4], [2, 3]];

print(simplifyRanges(ranges));
// => [[1, 4]]
```
